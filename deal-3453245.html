02 · Core explanation 交易实质

Существо операции

В данной модели ВЕСТ-ИСТ ТРЕЙД выступает продавцом товара. Входящий платёж является оплатой за товар по договору купли-продажи / поставки, а не агентским платёжным поручением, не переводом средств за третьих лиц и не платёжной услугой.

交易性质 · 商品销售合同项下的货款结算

Назначение поступления и его правовое основание определяются договором купли-продажи (поставки), инвойсом и спецификацией. Передача товара покупателю осуществляется на территории КНР, дальнейшая логистика — на стороне покупателя.

Документальное обеспечение операции

05Реквизиты и данные покупателя.

07Назначение платежа по договору купли-продажи / поставки в RUB.
08Сопроводительная переписка и иные подтверждающие материалы — по запросу банка.

04 · Compliance clarification 合规说明

Пояснения для банковского контроля

Статус

ВЕСТ-ИСТ ТРЕЙД не выступает платёжным агентом и не оказывает услуг платёжного посредничества.

Назначение средств

Поступающие средства не принимаются для последующего перевода третьим лицам.

Основание платежа

Платёж имеет товарное основание: договор купли-продажи / поставки + инвойс + спецификация.

Документы

Полный комплект документов по операции предоставляется по запросу банка.

Решение

Окончательное решение по операции принимает банк в рамках собственных процедур комплаенс- и валютного контроля. Настоящий документ носит исключительно пояснительный характер.

Notice

Документ носит информационный характер и не является гарантией проведения операции банком. Банк самостоятельно принимает решения в рамках собственных процедур комплаенса. Подготовлен для предварительного обсуждения со специалистом банка / менеджером по работе с клиентом и не предназначен для распространения.
